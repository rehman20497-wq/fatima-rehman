## 🛠️ Tech Stack & Architecture

- **Framework**: React 19 (Functional Hooks & Components)
- **Bundler**: Vite 6 (Optimized for lightning-fast speeds and clean static assets output)
- **Styling**: Tailwind CSS v4 (Using modern custom-designed design tokens)
- **Animation**: Motion (Framer Motion)
- **Audio Engine**: Custom HTML5 Web Audio API synthesizer wrapping romantic pads, heartbeats, and sparkles
- **Visualization**: HTML5 Canvas with optimized animation loops and particle systems

## 🚀 Local Development

To run the project locally on your machine, follow these simple steps:

### 1. Install Dependencies
```bash
npm install
```

### 2. Run the Development Server
```bash
npm run dev
```
This boots up the development server. Open [http://localhost:3000](http://localhost:3000) in your browser.

### 3. Build for Production
```bash
npm run build
```
This generates a production-ready, highly optimized `dist/` directory containing all static files.

## ☁️ Deployment Instructions

### Deploying to Vercel 🚀

This project is fully optimized for **Vercel** out-of-the-box as a modern React Single Page Application (SPA).

1. **Push your code to GitHub**:
   Create a new GitHub repository and push your project files.
   ```bash
   git init
   git add .
   git commit -m "feat: initial commit for Aik Khuwaab Tumhare Sath"
   git remote add origin <your-github-repo-url>
   git branch -M main
   git push -u origin main
   ```

2. **Deploy on Vercel**:
   - Go to [Vercel Dashboard](https://vercel.com) and click **Add New** > **Project**.
   - Import your GitHub repository.
   - Vercel will automatically detect **Vite** as the framework.
   - Click **Deploy**!
   - In less than a minute, your romantic application will be live on a fast, global CDN with an elegant HTTPS URL!
